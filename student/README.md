# Flower Shop System

> **Note**: This is a school project.

This project is a PHP-based flower shop web application with storefront, user registration, dashboards, product management, cart checkout, and payment integrations.

## Quick Start

1. Import your SQL schema into a local MySQL database.
2. Update the database settings in `db.php`.
3. Deploy the project folder to XAMPP or another PHP server.
4. Run `composer install` to install dependencies.
5. Open the project in your browser.
